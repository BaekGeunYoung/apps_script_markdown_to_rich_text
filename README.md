# Google Sheets Markdown to Rich Text Converter

Google Sheets에서 마크다운 텍스트를 서식이 적용된 리치 텍스트로 변환하는 Google Apps Script입니다.

## 기능

- **헤딩 변환**: H1(`#`), H2(`##`), H3(`###`)을 각각 다른 폰트 크기로 변환
- **볼드 텍스트**: `**텍스트**` 형식을 굵은 글씨로 변환
- **테이블 변환**: 마크다운 테이블을 Google Sheets 테이블로 변환 (테두리, 헤더 배경색 포함)
- **수평선 처리**: `---` 구분선 인식

## 설치 방법

1. Google Sheets 문서를 엽니다
2. **확장 프로그램** > **Apps Script** 클릭
3. `code.gs` 파일의 내용을 복사하여 붙여넣기
4. 저장 후 스프레드시트를 새로고침

## 사용 방법

1. 마크다운 텍스트가 있는 셀을 선택
2. 상단 메뉴에서 **Markdown** > **Convert to Formatted Text** 클릭
3. 선택한 셀부터 서식이 적용된 텍스트로 변환됨

## 지원하는 마크다운 문법

| 문법 | 설명 | 결과 |
|------|------|------|
| `# 제목` | H1 헤딩 | 14pt 볼드 |
| `## 제목` | H2 헤딩 | 13pt 볼드, 위아래 빈 행 추가 |
| `### 제목` | H3 헤딩 | 12pt 볼드 |
| `**텍스트**` | 볼드 | 굵은 글씨 |
| `\| A \| B \|` | 테이블 | 테두리가 있는 표 |
| `---` | 수평선 | 빈 행으로 처리 |

## 예시

**입력 (마크다운):**
```markdown
# 프로젝트 개요

## 주요 기능

**중요한** 기능입니다.

| 항목 | 설명 |
|------|------|
| A | 첫 번째 |
| B | 두 번째 |
```


**결과:**
- "프로젝트 개요"는 14pt 볼드로 표시
- "주요 기능"은 13pt 볼드로 표시되며 위아래 빈 행 추가
- "중요한"은 굵은 글씨로 표시
- 테이블은 테두리가 있는 표로 변환되며 헤더 행은 회색 배경
