MarkdownToRichText

Convert Markdown to Rich Text in Google Sheets

About

MarkdownToRichText is a Google Sheets add-on that converts Markdown text into formatted rich text. Write in Markdown and instantly see bold, italic, strikethrough, and other formatting applied directly in your spreadsheet cells.

Features

Bold & Italic - Convert **bold** and *italic* syntax
Strikethrough - Support for ~~strikethrough~~ text
Easy to Use - Simple menu integration in Google Sheets
Privacy Focused - All processing happens locally in your browser

How to Use

  1. Install the add-on from Google Workspace Marketplace
  2. Open a Google Sheets document
  3. Select cells containing Markdown text
  4. Go to Extensions > MarkdownToRichText > Convert
